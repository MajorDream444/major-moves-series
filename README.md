# Major Moves Series

The Major Moves Series is the publishing engine for the Major AI Mindset ecosystem.

It turns visual manifestos into short e-books, workbooks, Substack essays, Gumroad products, NotebookLM learning packs, podcasts, YouTube lessons, social assets, and community challenges.

## Core Principle

One visual seed can become a full doctrine, product, and learning system when the architecture is clear.

## Platform Roles

| Layer | Platform | Responsibility |
|---|---|---|
| Layer of Truth | GitHub | Version history, manuscripts, prompts, templates |
| Writing Room | Claude | Long-form drafting, expansion, refinement |
| Orchestration | ChatGPT | Strategy, systems architecture, curriculum, automation |
| Production | Airtable | Editorial pipeline and launch tracking |
| Knowledge | Notion | Research, reference, editorial library |
| Asset Vault | Google Drive | Covers, exports, audio, video, PDFs |
| Study + Audio | NotebookLM | Study guides, FAQs, audio discussions |
| Signal Layer | Substack | Audience validation and essays |
| Monetization | Gumroad | Finished products, bundles, workbooks |
