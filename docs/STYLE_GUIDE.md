# Major Moves Style Guide

## Voice

Direct. Grounded. Visionary. Practical. Human.

The reader should feel like someone finally explained the system without trying to impress them.

## Sentence Rhythm

Use short paragraphs.
Use clean transitions.
Let key lines breathe.

## Avoid

- Corporate language
- Academic padding
- Generic AI phrasing
- Empty motivation
- Overexplaining
- Copying Napoleon Hill language directly

## Use

- Clear doctrine
- Real-world founder language
- Systems thinking
- Sovereignty language
- Legacy framing
- Reflection prompts
- Major Move action steps

## Core Phrase Bank

- Build systems, not pressure.
- One clear step changes everything.
- The image is the door. The book is the doctrine.
- If everything stops when you stop, you are the system.
- Build once. Scale forever.
- Systems create freedom.
